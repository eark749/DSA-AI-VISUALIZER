Overview
Why Aho-Corasick?

Naive scanning checks each banned pattern separately — O(m × k) where k is blacklist size. Aho-Corasick precompiles all patterns into one automaton — O(m) total, independent of how many patterns exist.

🌲
Trie construction
All banned patterns inserted into a prefix trie. Each node stores children for next character. Terminal nodes mark a complete banned pattern — O(sum of pattern lengths) to build.
⚡
Failure links (Aho-Corasick key)
On mismatch, failure links jump to the longest proper suffix that is also a prefix in the trie. No backtracking. Enables scanning in a single O(m) pass over the text.
🔴
Streaming-safe
State carries across token boundaries. If a banned phrase spans token 47 and 48, the automaton catches it. Each new token continues from where the last left off — no window re-scan needed.

Python · Aho-Corasick
import ahocorasick

# Build automaton from blacklist
A = ahocorasick.Automaton()
blacklist = ["hack", "bomb", "kill",
             "malware", "exploit"]

for i, word in enumerate(blacklist):
    A.add_word(word, (i, word))
A.make_automaton()  # builds failure links

def scan_stream(text):
    # O(m) — single pass, any pattern
    for end_idx, (_, word) in A.iter(text):
        start = end_idx - len(word) + 1
        yield {
            "word": word,
            "pos": (start, end_idx)
        }

# Stream token-by-token
state = 0
for token in llm_stream():
    hits = list(scan_stream(token))
    if hits:
        block_output(hits)
    else:
        emit(token)

Key insights
What engineers get wrong
KEY INSIGHT — Aho-Corasick alone isn't enough
Pattern matching blocks known bad strings but can't detect novel jailbreaks, adversarial prompts, or context-dependent harm. "Ignore previous instructions" has no banned keyword. Real production guardrails layer Aho-Corasick (fast, zero false negatives on known patterns) with an ML classifier (contextual, catches novel attacks).
KEY INSIGHT — Stream scanning latency
Aho-Corasick shines on streams: O(m) where m = text length, regardless of pattern count. A naive approach (re.search per pattern) is O(m × p) — 10 patterns = 10× slower. For streaming output with 50 banned patterns, Aho-Corasick adds <0.1ms; naive regex adds ~5ms per chunk.
Automaton construction
Trie + failure links (BFS)
1. Insert all patterns into trie (prefix tree)
2. BFS from root: for each node, compute failure link = longest proper suffix that is also a prefix of another pattern
3. Compute output links: which patterns match at each node
4. At scan time: follow char transitions, fall back via failure links on mismatch
Production layers
Layer 1: Aho-Corasick — banned keywords, PII regex (SSN, card numbers)
Layer 2: ML classifier — intent/toxicity (OpenAI moderation API, Perspective API)
Layer 3: LLM self-check — "Does this response comply with policy?" (slow, expensive, for high-stakes only)
Layer 4: Output filtering — redact matched spans
Alternatives
ML moderation classifier
Fine-tuned BERT/DistilBERT on policy violation dataset. Scores intent/toxicity/PII. ~5–20ms. Handles paraphrased or encoded attacks.
Use when: novel attacks, context-dependent harm, multilingual
Prompt shields (Azure)
Dedicated API for jailbreak + indirect prompt injection detection. Runs before model. Returns risk scores per category (violence, hate, sexual, self-harm).
Use when: Azure stack, regulated industries, need audit trail
Constitutional AI (self-critique)
Model critiques its own output against a constitution (rules). Revises if violation found. Anthropic Claude trained this way. Expensive (~2× tokens) but captures nuanced harms.
Use when: nuanced policy, can afford latency, LLM-as-judge acceptable
Real-world systems
OpenAI Moderation API
Free ML classifier. 11 harm categories. Returns per-category scores + flagged bool. Integrated into ChatGPT input/output pipeline.
Llama Guard
Open-source safety classifier from Meta. Runs as a separate model call. Customizable policy taxonomy. Used in self-hosted LLM deployments.
NeMo Guardrails
NVIDIA's programmable guardrail framework. Colang language defines input/output rails as rules. Wraps any LLM. Aho-Corasick used for exact pattern matching layer.
Perspective API
Google Jigsaw toxicity scorer. TOXICITY, SEVERE_TOXICITY, IDENTITY_ATTACK, THREAT scores. Used by Discord, NYT comments for moderation pipelines.
